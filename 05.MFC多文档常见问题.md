## 多文档关闭view的标签页

在mainfrm.cpp中的CMainFrame::OnAppLook函数里删除：

```c++
//  mdiTabParams.m_bTabIcons = TRUE;//关闭view的标签页
//  EnableMDITabbedGroups (TRUE, mdiTabParams);//关闭view的标签页
```

改为

```c++
    mdiTabParams.m_bTabIcons = FALSE;
```

## 多文档中控制标题不显示文档名

```c++
    cs.style &= ~FWS_ADDTOTITLE;//此句控制标题不让他显示文档名
```
