# Markdown语法学习

Markdown是一种可以使用普通文本编辑器编写的标记语言，具有简单的语法格式，可以使普通的文本内容具有一定的格式，使编写出来的文档容易阅读且较为美观。  
Markdown的语法简洁明了、学习容易，而且功能比纯文本更强。因此有很多人用它写博客，目前主流博客平台都能较好的支持markdown。并且还常用于编写文档，将文档文件以“README.md”命名保存与软件根目录下。  
作为一名软件开发人员，可见学习markdown是多么重要。由于需要用的时候不那么频繁，对markdown的学习也一直断断续续，为了加强记忆，在此记录一下markdown的一些通用语法，提升自己，方便他人。

## 标题

Markdown语法：  
使用`#`号标记,可以表示1-6级标题，一级标题对应一个`#`号，二级标题对应两个`#`号，以此类推。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

效果：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 字体

Markdown可以给文字赋予三种字体，分别是斜体、粗体、粗斜体。  
语法：分别用1、2、3个 `*` 号或 `_` 将目标文本包含住。

```
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___
```

效果：

*斜体文本*  
_斜体文本_  
**粗体文本**  
__粗体文本__  
***粗斜体文本***  
___粗斜体文本___  

## 段落

### 换行

Markdown换行有两种方法，一种是在段尾使用两个及以上空格加回车表示段落的换行，另一种是使用一个空行即段尾两个回车表示重新开始一个段落。

效果：

两个空格加回车（段落内换行）：  
第二段。

两个回车（重新开始一个段落）：

第二段。

### 缩进（空格）

不同特殊占位符所占空白是不一样大的，常用全角 `&emsp;` 。

```markdown
&emsp; 或 &#8195; //全角
&ensp; 或 &#8194; //半角
&nbsp; 或 &#160;  //半角之半角
```

效果：

&emsp;&emsp;全角两个  
&ensp;&ensp;半角两个  
&nbsp;&nbsp;半角之半角两个

### 分割线

分割线可以用三个或三个以上的 - 或 * 都可以表示。

语法：

```markdown
---
----
* * *
*****
```

效果：

---
----
* * *
*****

### 删除线

要加删除线的文字左右分别用两个\~\~号包起来。

语法：

```markdown
~~这是加删除线的文字~~
```

效果：

~~这是加删除线的文字~~

### 下划线

下划线可以通过 HTML 的 \<u\> 标签来实现：

```markdown
<u>带下划线文本</u>
```

效果：

<u>带下划线文本</u>

## 图片

表示图片的语法如下：

```markdown
![avatar](路径)
//简单格式

![图片alt](路径 ''图片title'')
//图片alt 就是图片的替代文字，相当于对图片内容的解释。
//图片title 是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加

![avatar](./附件/00/攀岩者.jpg)
![石原里美](https://github.com/lovevantt/Depot/blob/master/pic/%E7%9F%B3%E5%8E%9F.jpg?raw=true "石原里美")
```

效果：

![avatar](./附件/00/攀岩者.jpg)

![石原里美](https://github.com/lovevantt/Depot/blob/master/pic/%E7%9F%B3%E5%8E%9F.jpg?raw=true "石原里美")

## 链接

链接有两种表示语法：

```markdown
[链接名称](链接地址)
或
<链接地址>
```

效果:

[lovevantt (范小猫)](https://github.com/lovevantt)  
或  
<https://github.com/lovevantt>

## 引用（区块）

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号，可以多层嵌套，文字加入多个 > 符号。  
区块

```markdown
> 第一层引用
>> 第二层引用
>>>> 第n层引用
```

效果：

> 第一层引用
>> 第二层引用
>>>> 第n层引用


## 列表

### 无序列表

语法：使用星号(*)、加号(+)或是减号(-)作为列表标记，后面加一个空格，然后后面加内容。

```markdown
- 列表内容
+ 列表内容
* 列表内容
```

效果：

- 列表内容
+ 列表内容
* 列表内容

### 有序列表

语法：数字加点加空格

```markdown
1. 列表内容
2. 列表内容
3. 列表内容
```

效果：

1. 列表内容
2. 列表内容
3. 列表内容

### 任务列表（前面有勾选框）

语法：减号(-)加空格( )加两个方括号(\[ \]或\[x\]),部分编辑器不支持。
```markdown
- [ ] 任务一 未做任务
- [x] 任务二 已做任务
```

效果：

- [ ] 任务一 未做任务
- [x] 任务二 已做任务

## 代码

### 行内代码

语法：将代码左右各用一个 ` 括起来。

```markdown
这个`function`是行内代码。
```

效果：

这个`function`是行内代码。

### 一行或多行代码

语法：行前和行后各用三个 ` 符号(即 ``` )包围起来。

```markdown
    ```c++
    E = mc
    E = mc^2
    ```
```

效果：

```c++
E = mc
E = mc^2
```

## 表格

语法：  
使用 `|` 来分隔单元格，使用 `-` 来分隔表头和内容。  
一般在第二行设置每一列的对其方式。 `-:` 表示标题栏和内容**右对齐**， `:-` 表示标题栏和内容**左对齐**，`:-:`或 `:-:` 表示标题栏和内容**剧中对齐**。

```markdown
左对齐 | 右对齐 | 居中对齐 | 未知
:- | -: | :-: | :-:
单元格 | 单元格 | 单元格 | 单元格
单元格 | 单元格 | 单元格 | 单元格
```

内容：

左对齐 | 右对齐 | 居中对齐 | 居中对齐
:- | -: | :-: | :-:
1 | 2 | 3 | 4
1 | 2 | 3 | 4

## 顺序图或流程图

语法：

```markdown
    ```sequence
    张三->李四: 嘿，小四儿, 写博客了没?
    Note right of 李四: 李四愣了一下，说：
    李四-->张三: 忙得吐血，哪有时间写。
    ```

    ```flow
    st=>start: 开始
    e=>end: 结束
    op=>operation: 我的操作
    cond=>condition: 确认？

    st->op->cond
    cond(yes)->e
    cond(no)->op
    ```
```

效果：

```sequence
张三->李四: 嘿，小四儿, 写博客了没?
Note right of 李四: 李四愣了一下，说：
李四-->张三: 忙得吐血，哪有时间写。
```

```flow
st=>start: 开始
e=>end: 结束
op=>operation: 我的操作
cond=>condition: 确认？

st->op->cond
cond(yes)->e
cond(no)->op
```

## Math格式

语法：使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现。

```latex
$$
x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

效果：

$$
x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
